# Universal Rules — gd-shared-update-docs-project-standard

These rules apply to **every step** of the skill. Each step file references this document instead of repeating them.

---

- 📖 **Read first:** Read the entire step file completely before taking any action.
- ⏸️ **Halt for input:** ALWAYS halt and wait for user input before proceeding to the next step or launching any subagent.
- 🔄 **Unrecognized input:** For unrecognized input or questions: answer the query, then redisplay the current menu.
